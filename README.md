# Queemma Sales Performance Analysis

## Business Intelligence & Data Analytics Project

This project analyzes Queemma company's sales data to uncover business insights, identify sales trends, and support data-driven decision-making.

### Key Insights
- Total Revenue: 23,377,525
- Best Performing Product: Laptop
- Strongest Region: East
- Top Salesperson: Kapnan
- Seasonal demand patterns identified

### Business Recommendations
- Increase marketing for top-performing products
- Train low-performing sales teams
- Expand operations in strong regions
- Improve distribution channels
- Introduce sales incentives

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

### Project Focus
- Sales Performance Analysis
- Regional Analysis
- Product Revenue Analysis
- Business Intelligence Dashboard
- Data Visualization
